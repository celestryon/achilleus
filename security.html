Before Submitting

ржЬржорж╛ ржжрзЗржУржпрж╝рж╛рж░ ржЖржЧрзЗ

  • Remove Metadata: If you're submitting documents, remove all metadata first. Many file types (PDFs, Word docs, photos) contain hidden information about who created them and when.
